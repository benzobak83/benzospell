Бесплатный сервис по распознаванию текста из изображений

и его дальнейшая проверка на правила орфографии :)

Загрузите файл формата JPG и распознайте содержащийся в нём текст для дальнейшей работы с ним

Выберите или перетащите файл с изображением для дальнейшего распознавания

Максимальный размер файла 2 Mb

Принцип работы ресурса
Отсканируйте или вставьте текст
Загрузите файл если нужно распознать текст
Выберите язык содержимого текста в файле
После обработки файла, получите результат
  • Форматы файлов
  • Изображения: jpg, jpeg, png
  • Мульти-страничные документы: pdf
  • Сохранение результатов
  • Чистый текст (txt)
  • Adobe Acrobat (pdf)
  • Microsoft Word (docx)
  • OpenOffice (odf)

Наши преимущества

  • Легкий и удобный интерфейс
  • Мультиязычность Сайт переведен на 9 языков
  • Быстрое распознавание текста
  • Неограниченное количество запросов
  • Поддержка 35+ языков распознавания текста
  • Защита данных. Данные между серверами передаются по SSL + автоматически будут удалены
  • Отсутствие регистрации
  • Использование движка Tesseract OCR
  • Распознавание области изображения (в разработке)
  • Обработано более чем 30.6M+ запросов
Основные возможности
Распознавание отсканированных файлов и фотографий, которые содержат текст
Получение напечатанного текста
Форматирование бумажных и PDF-документов в редактируемые форматы
Онлайн-редактирование документа
